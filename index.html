    Precursor to modern web development

    AJAX
  • Asynchronous JavaScript and XML.
  • Enabled update of web pages without the need for a full page reload

    • JSON
  • JavaScript Object Notation
  • JSON is based on a subset of the JavaScript programming language
  • Often used to transmit data between a server and a web application, as an alternative to XML

    First Major Front-end Library


    jQuery
  • Provided a consistent API for interacting with the DOM, regardless of the browser
  • Provided a unified event handling API that worked consistently across all browsers
  • Provided a simple and consistent API for making AJAX requests
  • Provided a simple and easy-to-use API for animating elements on a web page
  • Introduced css like selector support to grab dom nodes - eventually becoming a web standard

    Client-side Javascript frameworks emerged

  • Introduction of MVC pattern to the front end
  • Provided seperation of business logic from presentation logic
  • Popular frameworks backbone.js, ember.js, vue.js, angular.js
  • Templating libraries emerged proving capabilites such as conditional rendering, looping, and partials
  • Popular templating libraries Handlebars.js, Mustache.js

    Modern Front End Development

  • Singe Page Applications becoming popular
  • Front End tooling ecosystem emerged with popularity of Node.Js
  • Linting (eslint)
  • Transpiling (Babel.js, post-css)
  • Bundling (webpack, vite.js)
  • Minifying (uglify-js, Terser)
  • Packaging (npm, pnpm, yarn)
  • Popular libraries React.js, Vue.js, Svelte, Angular.js

    React Properties

  • React enables developers to write declarative UI
  • React is built around the concept of reusable components, which are small, self-contained UI elements.
  • Components can be composed together to create complex UIs and changes to one component will not affect the others
  • React is highly customizable allowing developers to allows developers to extend the library to suit their needs
  • React supports server side rendering, crucial for SEO
  • React is highly unopinionated, so manging application state, server state, routing etc is left to the developers
